PPM is a better web chat app that allows you to communicate with your friends without install any native app.
You only need a google account and you're ready to go!

Live Demo: https://paperplanemessanger.firebaseapp.com/

To run ppm in local machine, you need Nodejs and NPM.

1, run 'npm install' to install npm packages
2, run 'firebase serve' to run the local server


Please visit https://github.com/gjuoun/PPM for the newest version of PPM.
